Plaid Pattern Generator

Background

About

This is a program allowing the user to create plaid patterns. It was made largely to practice javascript.

Technical information

As stated, this is a javascript application, with limited use of jQuery (though I intend to convert it to vanilla javascript). It uses the built-in canvas API. The individual lines in the pattern have limited opacity to allow the intersecting portions to blend.

Room for improvement

I believe the jQuery used is a litte sloppy and kind of shoe horned in. For this reason, I intend to make it vanilla. Additionally, a more realist affect could be achieved by making the lines comprised of smaller lines to look like stitching. Beyond this, I think the javascript can be be trimmed down significantly.

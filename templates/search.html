'Random' Episode Picker

Title:

About

This page is intended to replicate the feeling of tuning into a network show to get a random repeat. While other sites exist for this function, this one contains what are, to my knowledge, unique features. Once you search your show and select it from the options provided, you exclude seasons by use of the check boxes and determine how much extra preference should be given to highly rated episodes by use of the "rating factor" sliding input. Once you provide these details, your episode will be generated.

Technical information

This page is built with Flask and uses the requests and BeautifulSoup modules to access pages on the IMDB website and extract the relevant data.

Once all needed information is accessed, it then generates a list of all relevant episodes receiving additional entries based on their rating. It then uses the choices function from the random module to select an episode from this list and returns details (again scraped from IMDB) to the user.

This was done instead of using the IMDB API for 2 reasons. First off, my understanding is that the IMDB API is poorly documented, meaning I would be likely to hit many road blocks without clear answers available. Additionally, I thought this was an excellent opportunity to learn web scraping and gain proficiency with tools I could use in many projects, as opposed to the limited possibilities of the IMDB API. While it would likely have been more efficient, this was undertaken as a learning opportunity and I believe this route was a better learning experience.

About this Site

My portrait was painted by a personal friend, Alex Hand. The icons used are from icons8.com. Navigation icon links can be found at the bottom of the navigation bar. The anchor can be found at the bottom of the page. The backgrounds for the content and the navigation bar are from pixabay.com. All custom fonts were acquired through fontsquirrel.com. The Episode Picker Project uses IMDB data.


Beyond the above, everything on this site is made by me. This whole site is deployed through Heroku and uses a Flask backend. As of now, all CSS is vanilla. Some images are editted using GIMP. The interface for Plaid Generator is made using React.js and create-react-app and displays through the Canvas API. Episode Picker is built using BeautifulSoup4 and Python 3.
